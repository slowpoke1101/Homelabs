\# My Homelab Journey
A record and blog of my homelab journey, from my first Raspberry Pi server to building an enterprise‑style network and security lab.

#Overview
This repository contains my complete homelab journey, including:

\- building and hardening my first server (mimiserver)

\- rehearsing a new build in VirtualBox and test environment (mimilab3)

\- creating an enterprise-style multi-network environment (VMware)

\- practicing security and enumeration (VMware and dvwa)

\- rebuilding my homenetwork with a dedicated router and NAS (flintnet)


#Structure
Each folder contains a project README with an explanation/blog entry, file key, screenshots, raw chronological notes, configs, and troubleshooting steps.
